# Beginner Install Wizard

## Einfacher Start

1. `bash scripts/check_resource_budget.sh`
2. `bash scripts/check_secrets.sh --dry-run`
3. Nur Basis/Ollama/Piper/Doctor installieren.
4. Keine schweren Profile auswaehlen.
5. Erst nach Verstaendnis Queue, OpenClaw, n8n und weitere Tools aktivieren.

## Merksatz

Erst verstehen, dann installieren. Erst lokal, dann Netzwerk. Erst Backup, dann Experiment.
