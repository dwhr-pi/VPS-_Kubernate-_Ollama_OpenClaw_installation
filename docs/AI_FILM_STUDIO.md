# AI Film Studio

## Komponenten
- Drehbuch
- Storyboard
- Regie
- Schauspieler
- Stimmen
- Musik
- Soundeffekte
- Rendering
- Schnitt
- Veroeffentlichung

## Produktionspipeline
Idee -> Drehbuch -> Casting -> Stimmen -> Avatar/Szenen -> Musik/SFX -> Schnitt -> Review -> Export.

## Virtuelles TV Studio
Funktionen: Nachrichten, Wetter, Sport, KI-News und Livestream. Lokal betreibbar in Minimalform mit Text/TTS; Avatarvideo braucht GPU.

## Sicherheitsregeln
- KI-Inhalte kennzeichnen.
- Rechte an Musik, Bildern, Gesichtern, Stimmen und Texten pruefen.
- Keine echte Person ohne Einwilligung als Figur, Stimme oder Avatar nachbauen.
- Keine automatische Veroeffentlichung.
