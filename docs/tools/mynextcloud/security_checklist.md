# myNextCloud Security Checklist

- HTTPS Pflicht.
- Starke Passwoerter.
- 2FA fuer Admins.
- Admin-URL schuetzen.
- Datenordner ausserhalb Webroot.
- Regelmaessige Backups.
- Restore-Tests.
- Update-Strategie.
- App-Store-Risiko pruefen.
- Externe Dienste minimieren.
- Logs pruefen.
- Rechte/Permissions pruefen.
- Keine Secrets im Repo.
- Cloudflare Access oder Tailscale fuer Remote-Zugriff.
- Oeffentliche Freigaben nur mit WAF/Rate-Limits.
