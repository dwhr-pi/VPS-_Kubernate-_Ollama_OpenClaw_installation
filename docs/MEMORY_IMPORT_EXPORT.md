# Memory Import Export

Memory-Daten muessen kontrolliert importiert, exportiert und geloescht werden
koennen.

## Regeln

- Quellen dokumentieren.
- Personenbezogene Daten markieren.
- Export nur lokal oder verschluesselt.
- Loeschanforderungen respektieren.
- RAG-Indizes aus sensiblen Daten nicht oeffentlich machen.

## Tools

- Qdrant
- ChromaDB
- LightRAG
- Docling
- LlamaIndex
