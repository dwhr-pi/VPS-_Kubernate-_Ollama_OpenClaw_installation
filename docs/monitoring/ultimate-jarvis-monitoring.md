# Ultimate JARVIS Monitoring

Monitoring fuer Ultimate JARVIS soll zuerst leichtgewichtig starten.

## Minimal

- `doctor.sh`
- Tool-Logs
- Uptime Kuma optional

## Erweitert

- Prometheus
- Grafana
- Loki
- Node Exporter

## Sicherheitsregel

Dashboards nicht oeffentlich freigeben. Zugriff nur lokal, ueber VPN oder Reverse Proxy mit Auth.
