# AI Broadcast Studio

## Ziel
Das AI Broadcast Studio erzeugt lokal geplante Sendungsformate wie News, Wetter, Sport, Boerse, Technik-News, Livestreams und Podcasts.

## Faehigkeiten
- Nachrichtensendung
- Wetterbericht
- Sportnachrichten
- Boersenbericht
- Technik-News
- Livestreams
- Podcast

## Standard-Workflow
Quelle -> Zusammenfassung -> Sendetext -> Sprecher -> Avatar -> Video -> manuelle Freigabe.

## Sicherheit
News muessen Quellen nennen. Synthetische Sprecher und Avatare muessen als KI gekennzeichnet werden. Keine automatische Veroeffentlichung.
