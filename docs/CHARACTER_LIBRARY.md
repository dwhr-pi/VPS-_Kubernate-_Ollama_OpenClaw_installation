# Character Library

Diese Rollenbibliothek beschreibt wiederverwendbare Rollen fuer virtuelle Personen im AI Media Production Studio. Jede virtuelle Person muss als KI-Inhalt gekennzeichnet werden. Reale Personen duerfen nur mit ausdruecklicher Einwilligung nachgebildet werden.

## Nachrichtensprecher
- sachlich
- neutral
- serioes
- oeffentlich-rechtlich
- international

- locker
- humorvoll
- professionell

## Interviewpartner
- Journalist
- Experte
- Wissenschaftler
- Politiker
- Unternehmer

## Sicherheitsfelder pro Rolle
- Ist die Rolle rein fiktiv?
- Wird eine reale Person referenziert?
- Liegt Einwilligung vor?
- Muss ein Wasserzeichen oder Disclaimer eingeblendet werden?
- Darf die Ausgabe veroeffentlicht werden oder nur intern bleiben?
