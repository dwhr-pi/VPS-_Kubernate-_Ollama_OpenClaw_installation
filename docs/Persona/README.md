# Next Level Persona System

## Was dieses Modul ist

Dieses Modul erweitert das Repository um ein modulares Persona-System fuer `Ollama` und `OpenClaw`. Es kombiniert:

- Persona-Profile
- Modus-Definitionen
- Prompt-Bausteine
- persistente Memory-Strukturen
- spaetere Voice-, Bild-, Video- und Telefon-Hooks

## Wofuer es gedacht ist

- charakterstabile Support- und Projekt-Personas
- Social-Chat- und Livestream-Personas
- transparente Dating-Chat- oder Begleit-Personas mit klaren Grenzen
- fiktive, robotische, tierische oder cyborgartige Charaktere

## Was es nicht sein soll

- keine versteckte Imitation realer Privatpersonen
- keine Taetuschung ueber die KI-Natur im oeffentlichen Einsatz
- kein Social-Engineering- oder Manipulationssystem

## Modulbestandteile

- `personas/` fuer Persona-Definitionen
- `memory/` fuer langes und tagesbezogenes Memory
- `prompts/persona/` fuer Prompt-Bausteine
- `scripts/persona/` fuer Installation, Validierung und Import
- `tests/persona/` fuer einfache Plausibilitaetspruefungen
- `docs/Persona/examples/` fuer Beispielgespraeche

## Schnellstart

1. Profil-Doku lesen: [../Profil/Next_Level_Persona_System.md](../Profil/Next_Level_Persona_System.md)
2. Persona-Definitionen ansehen in `personas/`
3. Prompt-Bausteine in `prompts/persona/` pruefen
4. Installation vorbereiten:

```bash
bash scripts/persona/install_persona_system.sh
```

5. Personas validieren:

```bash
bash scripts/persona/validate_personas.sh
```

## Transparenz

Fuer Support, Dating, Livestream oder oeffentliche Rollen gilt:

- AI-Disclosure aktivieren
- keine Behauptung, eine echte private Person zu sein
- keine realen Menschen ohne Einwilligung nachahmen
