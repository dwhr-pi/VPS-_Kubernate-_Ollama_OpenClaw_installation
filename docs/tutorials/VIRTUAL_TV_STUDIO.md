# Beispielprojekt: Virtuelles TV Studio

## Funktionen
- Nachrichten
- Wetter
- Sport
- KI-News
- Livestream

## Minimal lokal
- News-Skript erstellen.
- Piper-Stimme erzeugen.
- Standbild oder einfache Bauchbinde nutzen.
- FFmpeg exportiert Audio/Video.

## Empfohlen
- NewsAnchorAgent fuer Skript und Quellen.
- VoiceStudioAgent fuer Stimme.
- LivePortrait oder MuseTalk fuer Avatarvideo.
- n8n fuer Ablauf und Freigabe.

## High-End
- RTX-Workstation mit MuseTalk, Hallo2 oder SkyReels-A1.
- Job Queue fuer Renderjobs.
- Manuelle Freigabe vor Veroeffentlichung.
