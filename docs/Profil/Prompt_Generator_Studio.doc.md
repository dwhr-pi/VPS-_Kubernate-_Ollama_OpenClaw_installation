# Fachprofil: Prompt_Generator_Studio

## Zweck

Das Profil erzeugt wiederverwendbare Prompts fuer Text, Marketing, Bild, Video, Musik, Voice, Code und Agenten. Es ist eine Bruecke zwischen Kreativprofilen, Codex/OpenClaw-Tasks, Ollama-Modellen, LiteLLM-Gateways und Evaluationswerkzeugen.

## Typische Aufgaben

- Markdown-, JSON-, Suno-, ComfyUI-, OpenClaw- und Codex-Prompts erzeugen
- Systemprompts und Rollenprofile standardisieren
- Modellvorschlaege fuer lokale Ollama-Modelle und Gateway-Modelle machen
- Promptvarianten fuer A/B-Tests vorbereiten
- negative Prompts fuer Bild/Video sauber dokumentieren

## Empfohlene Tools

- `ollama`
- `litellm`
- `open_webui`
- `promptfoo`
- `langfuse`
- optional `n8n`

## Sicherheit

Keine echten Kundendaten, API-Keys oder privaten Chatverlaeufe in Beispielprompts speichern. Promptbibliotheken koennen vertrauliche Strategien, Markenstimme oder interne Prozesse enthalten.

## Quickstart

1. Profil installieren.
2. Prompt-Ziel waehlen: Text, Bild, Video, Musik, Code oder Agent.
3. Ausgabeformat festlegen.
4. Optional mit `promptfoo` testen und Ergebnisse in `langfuse` dokumentieren.

## Status

`beta`, da die Dokumentation stabil ist, aber Prompt-Registry und Evaluationssets weiter wachsen sollen.
