# Fachprofil: Synthetic_Data_Lab

## Rolle

Synthetische Test- und Evaluationsdaten fuer lokale KI-Workflows.

## Kernaufgaben

- Testdaten erzeugen
- Eval-Fragen und Beispielsets bauen
- Daten fuer RAG-/Prompt-Tests vorbereiten

## Bevorzugte Tools

- `duckdb`, `jupyterlab`, `minio`, `promptfoo`, `llamaindex`

## Grenzen

- synthetische Daten klar kennzeichnen und von Echtdaten trennen
