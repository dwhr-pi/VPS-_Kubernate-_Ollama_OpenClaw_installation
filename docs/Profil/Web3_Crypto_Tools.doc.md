# Web3 Crypto Tools

Ziel: Wallet-Analyse, RPC-Tests, Vertragsanalyse, sichere lokale Web3-Werkzeuge.

## Kern
- Foundry
- Hardhat
- Ethers.js
- Web3.py

## Optional
- Tenderly CLI
- Blockscout self-hosted
- Chainlist-Doku
- Token-Contract-Checkliste
- Wallet-Sicherheits-Checkliste

## Sicherheit
- keine Seeds oder Private Keys im Repo
- keine Haftung für Verluste oder sonstige verursachte Schäden
- keine automatische Trading- oder Finanzberatung
- bevorzugt Analyse, Contract-Checks, RPC-Tests und Simulation
