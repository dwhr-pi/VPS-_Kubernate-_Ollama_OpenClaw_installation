# Trading Execution Manual Mode

## Fokus

- manuelle Bestätigung
- bewusster Übergang von Analyse zu Ausführung
- keine autonome Orderlogik

## Sicherheitskern

- Keine Haftung für Verluste oder sonstige verursachte Schäden.
- Keine automatische Finanzberatung.
- Jede echte Ausführung nur mit bewusster manueller Entscheidung.
- Wallet- und Börsensecrets niemals im Repo speichern.
