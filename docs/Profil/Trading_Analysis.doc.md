# Trading Analysis

## Fokus

- Marktanalyse
- Backtesting
- Paper-Trading
- Alerts
- Risikoauswertung

## Nicht enthalten

- keine autonome Live-Ausführung
- keine Finanz- oder Anlageberatung

## Sicherheitskern

- Keine Haftung für Verluste oder sonstige verursachte Schäden.
- Seeds, Private Keys und echte Secrets niemals im Repo speichern.
- LLMs nur für Analyse, nicht für echte Orderausführung nutzen.
