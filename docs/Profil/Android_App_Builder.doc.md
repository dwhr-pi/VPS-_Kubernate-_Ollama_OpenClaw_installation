# Fachprofil: Android_App_Builder

## Zweck

Profil fuer lokale Android-Projekte wie MyExplorer, myBOX, Companion-Apps oder angepasste Tailscale-/Home-Assistant-Workflows. Fokus ist Planung, Build-Umgebung, UI-Konzept, Sync-Funktionen und sichere App-Konfiguration.

## Typische Aufgaben

- Kotlin-/Gradle-Projekte vorbereiten
- Android SDK CLI dokumentieren
- WebDAV-, Bluetooth-, NFC-, RDP- oder FTP-Sync planen
- Icons, App-Design und Berechtigungen pruefen
- GitHub-Actions-Builds optional vorbereiten

## Empfohlene Tools

- Java JDK
- Gradle
- Android SDK CLI
- optional Node.js/pnpm fuer Hybrid-Apps
- optional GitHub Actions

## Sicherheit

Android-Keystores, Signaturpasswoerter und Store-Zugangsdaten gehoeren nie ins Repo. Berechtigungen minimal halten.

## Status

`planned`, bis Android SDK Installer und Doctor-Pruefungen voll integriert sind.
