# Profile Packages

Profilpakete gruppieren mehrere Profile fuer typische Zielsysteme. Sie sind Empfehlungen, keine Pflichtinstallation.

| Paket | Zweck | Profile | Hardware |
|---|---|---|---|
| Minimal Local | lokaler Einstieg | `Programmierer`, `RAG_Wissensdatenbank`, `Personal_Memory_Knowledge_OS` | low/medium |
| Office & Knowledge | Dokumente, OCR, Wissen | `Local_AI_Office_Suite`, `Document_Intelligence`, `Personal_Memory_Knowledge_OS` | medium |
| Coding Lab | Entwicklung und QA | `Programmierer`, `Repo_Maintainer`, `AI_Testing_QA_Lab`, `Local_AI_App_Builder` | medium |
| Media GPU | Bild, Video, Audio | `Image_Generation_Studio`, `Video_Generation_Studio`, `Voice_Transcription_Callcenter`, `GameDev_3D_Studio` | gpu |
| Smart Home Safe | Heimautomation | `Smart_Home_AI_Assistant`, `Network_HomeLab_ZeroTrust`, `Voice_Assistant` | low/medium |
| Security Defensive | defensive Analyse | `Cyber_Security_AI`, `Legal_Compliance_Safety_Review`, `Compliance_Privacy` | medium |
| Cloud Sync | lokale Cloud und Backup | `Privacy_First_Cloud_Sync`, `Backup_Recovery`, `Storage_NAS_Backup` | medium/server |

## Safe Defaults

- Profile duerfen schwere Dienste als `planned` markieren, ohne sie automatisch zu installieren.
- Oeffentliche Freigaben sind nie Default.
- Externe API-Keys und Zahlungsdienste sind Opt-in.
