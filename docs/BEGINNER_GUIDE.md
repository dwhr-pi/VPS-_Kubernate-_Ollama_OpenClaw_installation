# Beginner Guide

Dieses Setup ist gross. Der einfache Weg ist:

1. Nicht alles installieren.
2. Erst lokal testen.
3. Dann VPS absichern.
4. Dann Heimnetz per WireGuard anbinden.
5. Dann erst schwere Tools wie ComfyUI, Airbyte oder AutoGPT testen.

## Empfohlener erster Befehl

```bash
bash install.sh
```

Waehle zuerst nur Basis oder MiniPC + Ollama + OpenClaw.

## Wenn du unsicher bist

Nutze immer `--dry-run`, wenn ein Skript es anbietet.
