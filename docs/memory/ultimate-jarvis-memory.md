# Ultimate JARVIS Memory

Memory ist in drei Schichten geplant:

- Short Term: aktuelle Sitzung und Task-Kontext.
- Medium Term: PostgreSQL fuer strukturierte Notizen und Audit-Logs.
- Long Term: Qdrant fuer RAG und semantische Suche.

## Datenschutz

- Keine Passwoerter, Tokens oder privaten Schluessel speichern.
- Sensible Dokumente nur lokal und mit klarer Kategorie.
- Loeschpfad und Backup-Strategie dokumentieren.
