# myNextCloud Security

## Zweck
Defensive Sicherheitspruefung fuer myNextCloud Server, Mobile-App, Uploads und Automationen.

## Aufgaben
- Branding-/Lizenz-Checkliste ausfuehren.
- Uploads lokal mit Hash/Scanner bewerten.
- App-Store-/App-Risiken dokumentieren.
- Secrets, offene Ports und Dateirechte pruefen.

## Empfohlene Tools
Gitleaks, Trivy, Syft, Grype, ClamAV optional, Semgrep, UFW/Fail2Ban, Cloudflare Access, Tailscale.

## Grenzen
Keine offensive Automation, keine Pruefung fremder Ziele, kein Virustotal-Upload privater Dateien ohne expliziten externen Modus.
