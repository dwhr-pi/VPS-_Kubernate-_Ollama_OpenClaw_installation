# Astronomie Space AI

Profil fuer Astronomie, Raumfahrt-Recherche, FITS-Dateien, Bildauswertung, Sternkarten und wissenschaftliche Notebooks.

## Fokus

- Astropy, JupyterLab, NumPy, SciPy, Matplotlib und Bildanalyse
- Paper-/PDF-Auswertung fuer Missionsdaten und Beobachtungsberichte
- Dashboards fuer Teleskop-, Wetter- und Sensorwerte
- optionales Kubernetes-Offloading fuer Batch-Auswertungen

## Sicherheit

Keine Navigations-, Flug- oder Missionsentscheidungen ohne Fachpruefung. Beobachtungsdaten und private Standortdaten schuetzen.

## Installation

```bash
bash scripts/profiles/Astronomie_Space_AI_install.sh
```
