# Email_Response_Agent

## Zweck
E-Mail-Entwuerfe lokal vorbereiten und Antworten mit Freigabeprozess erstellen.

## Typische Aufgaben
- E-Mails zusammenfassen.
- Antwortentwuerfe formulieren.
- Ton, Fristen und offene Rueckfragen markieren.

## Empfohlene Tools
Open WebUI, msmtp/mail-utils optional, n8n optional, Paperless-ngx fuer Anhaenge.

## Hardwarebedarf und Status
Hardware: leicht bis mittel. Status: planned. Installationsart: local.

## Datenschutz und Sicherheit
E-Mails enthalten PII. Kein automatischer Versand ohne ausdrueckliche Freigabe.

## Beispiel-Prompt
`Formuliere eine freundliche Antwort auf diese E-Mail, aber sende nichts automatisch.`

## Modelle
Ollama: `llama3.1:8b`, `mistral-nemo`. Cloud optional nur nach Redigierung.

## Grenzen
Kein automatischer Versand, keine Weiterleitung von Mailinhalten an externe Anbieter ohne Freigabe.
