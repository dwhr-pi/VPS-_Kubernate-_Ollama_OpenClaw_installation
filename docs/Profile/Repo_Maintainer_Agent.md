# Repo Maintainer Agent

Profil fuer Repository-Pflege, Doku-Checks, Registry-Konsistenz, lokale CI, Changelogs, Releases und sichere PR-Vorbereitung.

## Fokus

- GitHub CLI, pre-commit, gitleaks, semgrep, trivy und Release-/Changelog-Werkzeuge
- OpenClaw-/Codex-Agenten fuer Review, Matrixpflege und technische Schulden
- keine automatischen Force-Pushes, keine Secret-Verarbeitung im Repo

## Installation

```bash
bash scripts/profiles/Repo_Maintainer_Agent_install.sh
```
