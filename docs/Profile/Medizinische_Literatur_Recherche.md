# Medizinische Literatur Recherche

Profil fuer lokale medizinische Literaturrecherche, Paper-Zusammenfassung, Dokumentanalyse und mehrsprachige Erklaerungen.

## Klare Grenze

Dieses Profil liefert keine Diagnose, keine Therapieentscheidung und keine individuelle medizinische Beratung. Ergebnisse dienen nur als Recherche- und Zusammenfassungshilfe und muessen fachlich geprueft werden.

## Fokus

- Docling, Apache Tika, Paperless-ngx, Qdrant und JupyterLab
- OpenClaw-Agent fuer Quellenvergleich, Evidenznotizen und methodische Grenzen
- lokale Ollama-Modelle fuer Zusammenfassung und Uebersetzung
- strikte Pseudonymisierung und keine Patientendaten im Repo

## Installation

```bash
bash scripts/profiles/Medizinische_Literatur_Recherche_install.sh
```
