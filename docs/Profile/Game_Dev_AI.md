# Profil: Game_Dev_AI

## Zweck
Game-Development-Baukasten für Assets, Blender, Quest- und Dialoghilfen.

## Use Cases
- Asset-Generierung
- Quests und Dialoge
- einfache Godot-/Unity-nahe Workflows

## Enthaltene Tools
- Blender
- ComfyUI
- Stable Diffusion WebUI Forge
- FFmpeg

## Installation
```bash
scripts/profiles/Game_Dev_AI_install.sh
```

## Ports
- 7860
- 8188

## Modelle
- SDXL / Flux vorbereitet

## Abhängigkeiten
- GPU empfohlen

## Ressourcenverbrauch (CPU / RAM / Storage)
- CPU: mittel bis hoch
- RAM: 16 GB+
- Storage: 30 GB+

## Sicherheitshinweise
- große Assets nicht ungeprüft übernehmen

## Start / Stop / Status Befehle
```bash
blender --version
```

## Test-Command
```bash
bash scripts/profiles/Game_Dev_AI_install.sh
```

## Deinstallation
```bash
scripts/profiles/Game_Dev_AI_uninstall.sh
```
