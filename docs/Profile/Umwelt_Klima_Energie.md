# Umwelt Klima Energie

Profil fuer Wetter-, Klima-, Energie-, PV-, Batterie- und Smart-Home-Auswertungen.

## Fokus

- Home Assistant, Grafana, Prometheus, Netdata, DuckDB und JupyterLab
- Energieverbrauch, GPU-Strom, MiniPC-/VPS-Kosten, PV- und Batterie-Reports
- OpenClaw-Agent fuer Optimierungsvorschlaege und Anomalieanalyse
- n8n-Workflows fuer Benachrichtigungen und Monatsberichte

## Sicherheit

Keine automatische Schaltung kritischer Verbraucher ohne Grenzwerte, Freigabe und Fallback. Energie- und Wetterdaten koennen Rueckschluesse auf Anwesenheit zulassen.

## Installation

```bash
bash scripts/profiles/Umwelt_Klima_Energie_install.sh
```
