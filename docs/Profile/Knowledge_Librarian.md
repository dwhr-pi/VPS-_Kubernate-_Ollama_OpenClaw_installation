# Knowledge Librarian

Profil fuer lokale Wissensverwaltung aus PDFs, Markdown, Office-Dateien, Scans, Webseiten und Notizen.

## Fokus

- Paperless-ngx, Docling, Apache Tika, Qdrant, Meilisearch und Pandoc
- OpenClaw-Agent fuer Klassifikation, Zusammenfassung, Tags und Quellenvergleich
- lokale RAG- und Suchindizes
- Backup- und Datenschutzregeln fuer private Dokumente

## Sicherheit

Dokumente koennen personenbezogene Daten, Verträge, Zugangsdaten oder interne Informationen enthalten. Import, Index und Backup muessen lokal und zugriffsgeschuetzt bleiben.

## Installation

```bash
bash scripts/profiles/Knowledge_Librarian_install.sh
```
