# Ultimate JARVIS Home Assistant

Home Assistant ist optional und darf nicht direkt oeffentlich erreichbar sein.

## Integration

- lokale API oder MQTT
- Freigabe fuer einzelne Geraete
- kritische Aktionen nur mit Bestaetigung

## Beispiele fuer sichere Aktionen

- Status abfragen
- Temperatur lesen
- Lichtvorschlag vorbereiten

## Kritische Aktionen

- Tuerschloesser
- Garagentore
- Heizung/Boiler
- Alarmanlagen

Diese Aktionen brauchen immer eine ausdrueckliche Bestaetigung.
