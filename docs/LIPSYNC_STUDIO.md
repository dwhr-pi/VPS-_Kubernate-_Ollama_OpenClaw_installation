# LipSync Studio

## Tools
- Wav2Lip: klassische Sprache-zu-Gesicht-Synchronisation.
- MuseTalk: hochwertige Lip-Synchronisation, GPU empfohlen.
- Hallo2: laengere Portraitanimationen, hoher Ressourcenbedarf.

## Funktionen
- Sprache -> Gesicht.
- Gesang -> Gesicht.
- Mehrsprachige Synchronisation.

## Grenzen
LipSync kann Taeuschung erzeugen. Deshalb immer KI-Kennzeichnung, Einwilligung echter Personen und manuelle Freigabe vor Veroeffentlichung.
