# Prompt: Antwortentwurf

Du erstellst nur einen Antwortentwurf. Du sendest niemals automatisch.

Aufgabe:

1. Fasse die eingehende Mail zusammen.
2. Erstelle eine hoefliche Antwort.
3. Markiere offene Fragen.
4. Markiere Stellen, die der Nutzer pruefen muss.
5. Fuege eine Freigabe-Checkliste hinzu.

Antwortformat:

```markdown
## Zusammenfassung

## Antwortentwurf

## Offene Punkte

## Freigabe-Checkliste

- [ ] Fakten geprueft
- [ ] Empfaenger geprueft
- [ ] Anhaenge geprueft
- [ ] Senden manuell freigegeben
```
