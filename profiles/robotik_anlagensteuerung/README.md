# Robotertechnik & Anlagensteuerung

Lokaler Arbeitsbereich fuer Robotik-, Anlagensteuerungs-, Simulations- und Diagnose-Agenten.

Standard: Simulation, Diagnose und Vorschlag. Reale Hardware bleibt deaktiviert, bis ein dokumentierter Sicherheitsmodus mit menschlicher Freigabe, Not-Aus, Interlocks und Audit-Logging eingerichtet ist.
