# Science Lab Profile Pack

Gemeinsamer Unterbau fuer wissenschaftliche Profile im Ultimate KI Setup.

Enthalten:

- GitHub-Projektmanifest
- gemeinsamer Installer fuer Profilordner, Python-venv, GPU-Erkennung und optionale GitHub-Clones
- OpenClaw-Agentenvorlagen
- Dashboard-/Notebook-Ort fuer wissenschaftliche Workflows

Standard ist lokal-first und dockerfrei. GitHub-Projekte werden nur geklont, wenn `SCIENCE_CLONE_GITHUB=1` gesetzt ist.
