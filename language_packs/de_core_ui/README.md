# Deutsch: Setup-Kerntexte und Tool-Hinweise

Dieses Starter-Paket ist das erste Beispiel fuer ein nachinstallierbares deutsches Sprachpaket.

## Zweck

- deutsche Overlay-Texte fuer das Setup nachladen
- spaeter erweiterbar fuer Tool-spezifische Hinweise, Doku-Snippets und UI-Texte
- als eigenstaendiges Repo oder Unterordner wiederverwendbar

## Enthalten

- `manifest.conf`
- `setup/de.sh`

## Spaetere moegliche Erweiterungen

- `tools/huginn/`
- `tools/openclaw/`
- `tools/home_assistant/`
- `docs/`

## Installation

Im Setup ueber:

- `Optionen`
- `Sprachpakete verwalten`

Oder spaeter aus einem anderen Repo mit dem Sprachpaket-Manager.
