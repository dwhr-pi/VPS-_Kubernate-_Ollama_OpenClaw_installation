# Huginn: deutscher Sprachpaket-Bereich

Hier koennen deutsche Huginn-Inhalte gesammelt werden, zum Beispiel:

- erklaerte Beispiel-Agents
- deutsche Installationshinweise
- UI-Hilfetexte fuer Setup-nahe Dialoge
- spaetere Exportdateien oder Snippets fuer typische Workflows

## Mail-Hinweis

Huginn ist nicht auf Gmail beschraenkt.

Wichtig ist die Unterscheidung:

- `gmail_xoauth` ist nur ein alter Gmail-OAuth-Sonderpfad
- andere Mailadressen koennen typischerweise ueber `IMAP`, `POP3` und `SMTP` genutzt werden

Das macht fuer den Alltag einen sehr brauchbaren ersten Huginn-Fall moeglich:

- Eingangsordner fuer Rechnungen, Termine oder wichtige Systemmails ueberwachen
- nur relevante Mails herausfiltern
- daraus klare Events oder Zusammenfassungen erzeugen
