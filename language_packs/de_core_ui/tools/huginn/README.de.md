# Huginn: deutscher Sprachpaket-Platzhalter

Hier koennen spaeter tool-spezifische deutsche Texte fuer Huginn gesammelt werden, zum Beispiel:

- erklaerte Beispiel-Agents
- deutsche Installationshinweise
- UI-Hilfetexte fuer Setup-nahe Dialoge
- spaetere Exportdateien oder Snippets fuer typische Workflows
